# Классификация видов птиц по аудиозаписям

Автоматическое распознавание вида птицы по короткому акустическому фрагменту.

Авторы: студенты НИУ ВШЭ, Албегов Тамерлан и Горлова Анна

**Описание проекта**

Проект посвящён задаче многоклассовой классификации биоакустических сигналов птиц в условиях реальных полевых записей. В отличие от классических задач распознавания речи, биоакустические данные характеризуются высокой внутриклассовой вариативностью (индивидуальные различия особей, географические диалекты, сезонные вариации вокализации), значительным уровнем фонового шума, а также перекрытием частотных диапазонов между видами.

**Цель** — построить и сравнить несколько подходов к автоматической идентификации вида по короткому акустическому фрагменту, оценить применимость предобученных трансформерных моделей после доменоспецифичного fine-tuning, а также исследовать влияние акустической различимости классов и стратегии сегментации на качество классификации.

Данные собраны через Xeno-canto API v3 — краудсорсинговую базу орнитологических аудиозаписей. Для формирования выборки отбирались записи качества A и B по внутренней системе оценки платформы.

**Состав датасета**

15 видов птиц, распространённых в умеренной зоне Евразии
~175–200 записей на класс, суммарно ≈ 2973 аудиофайла
Каждая запись нарезается на 5-секундные сегменты (не более 5 на файл)
Разбивка: 70% train / 15% val / 15% test


**В проекте реализованы и сравниваются три подхода:**

1. CNN + SpecAugment 
2. CNN + активная сегментация (RMS energy)
3. AST (Audio Spectrogram Transformer)

**Ключевые наблюдения**
Кукушка и чёрный стриж — лучшие классы: акустически уникальные вокализации без пересечения с другими видами.
Зарянка, зяблик, чёрный дрозд — систематически путаются из-за перекрывающегося частотного диапазона (2–8 кГц).
Серая ворона: precision 0.91, recall 0.31 — высокая вариативность вокализации (крики тревоги, контактные, подражание) мешает стабильному распознаванию.
Сизый голубь: recall 0.89, низкая precision — модель избыточно предсказывает этот класс.
Переход с 15 на 10 акустически более различимых классов поднял F1 с 0.456 до 0.558 при той же архитектуре и объёме данных — сложность задачи определяется не только числом классов, но и акустической структурой пространства.

**Ограничения**
Ни одна из моделей не поддерживает класс «нет птицы» (open-set recognition)
Равномерная нарезка может включать фрагменты без целевой вокализации — именно это мотивирует эксперимент с RMS-сегментацией
Обучение велось в Google Colab, часть экспериментов (AST, активная сегментация) не была доведена до финальной оценки из-за прерывания сессии; код присутствует в ноутбуке в полном виде
